时间都去哪了

发表于 2018-03-24 | 分类于 生活

电影看完了，已经是晚上十一点。揉揉疲惫的双眼，突然间感觉哪里有点不对,为何自己这一天什么都没干，一天又过去了，然而，我真的什么都没干吗？

计划

在家一整天都去学习

实际

  1. 早上八点半左右醒来，躺着被窝里看nba，看到十一点左右
  2. 坐起来本来准备看书，结果却是刷手机，直到十二点多
  3. 出去溜达了一圈，吃了个饭，下午没有目的性的玩电脑或者手机直至四点
  4. 拿起书本看了大概两个小时，中间数次去查看手机
  5. 六点多去超市溜达一圈，回来煮了个面，然后抱着电脑看电影
  6. 看完电影已是十一点了，此时方突然醒悟，一天就这样过去了

时间都去哪了

今天，我本拥有十几个小时的自由支配时间，我曾计划全部用来学习，为何真正学习的时间只有两个小时，而且还是在三心二意的情况下，效率并不高。

我能说我我今天在家学习了吗？可以，因为我今天的计划就是学习，也确实抽出时间去学了。

然而真实情况是这样吗？自然不是，那不过是我自欺欺人罢了。

我学到了什么，是那两个小时看的书吗，我想不是的，如果我心无旁骛非常专注的去学习，一个小时的收获都会比这多。那中间的玩手机和电脑呢，我努力去回忆，却发现什么都记不起来了，碎片化的信息，来的快，忘的也快。

我似乎明白我的时间去哪了，我花了太多时间在计划之外的事情上，而且是没有任何目的性的事情，任由时间流逝，却不自知。

我该如何改变

既然决定一整天都去学习，那么在这十几个小时内至少要有八个小时是有效的学习时间，为自己定下以下目标

  1. 不在床上做除了睡觉之外的任何事情
  2. 闲暇之余刷手机的时间一次性不能超过十五分钟
  3. 尽量少做没有任何目的性的事情，例如长时间刷朋友圈，看短视频，漫无目的的看新闻
  4. 学习的时候远离手机，要保持专注，避免受到干扰

明天又会是全新的一天，加油！
